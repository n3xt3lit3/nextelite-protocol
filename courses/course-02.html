COURSE 02

Claude Code in Action

// the tool behind the protocol

The entire NextElite site was built with Claude Code — an AI coding agent that lives in your terminal. This course teaches you to master it. Not theory. Real workflows, real automation, real project work.

PROGRESS 0 / 8
LESSON 01
What is Claude Code?
Claude Code is Anthropic's CLI for Claude — an AI coding agent that lives in your terminal. It can read files, edit code, run commands, search codebases, and manage git. It's how the entire NextElite site was built. Available as CLI, desktop app, web app, and IDE extensions.
EXERCISE
Install Claude Code (npm install -g @anthropic-ai/claude-code) and run claude in your NextElite project directory. Ask it to describe the project structure.
LESSON 02
Architecture & Tool Use
How Claude Code works under the hood — tool integration (Read, Write, Edit, Bash, Glob, Grep), how it interacts with your codebase. The permission system controls what Claude Code can do without asking. Understanding tool calls helps you predict behavior and debug issues.
EXERCISE
Ask Claude Code to find all CSS color variables in the NextElite project. Watch which tools it uses (Glob, Grep, Read) and how it chains them together to get the answer.
LESSON 03
Context Management
CLAUDE.md files are project instructions that persist across sessions. They tell Claude Code about your project, your rules, your preferences. The memory system and conversation context determine how effectively the agent works. The right context turns a generic AI into your project partner.
EXERCISE
Create a CLAUDE.md in the nextelite-website root that describes the project, brand colors, design rules, and key decisions.
LESSON 04
Effective Workflows
How to work with Claude Code effectively. Start broad then refine. Use screenshots and images to communicate design intent. Review diffs before accepting changes. Git integration handles commits, branches, and PRs without leaving the terminal.
EXERCISE
Use Claude Code to add a new section to the NextElite site. Take a screenshot of a design you like and ask Claude Code to recreate it with the NextElite design system.
LESSON 05
Custom Slash Commands
Build custom commands with markdown files in .claude/commands/. Each file becomes a slash command. They can accept parameters, chain operations, and automate repetitive workflows. Turn multi-step processes into single commands.
EXERCISE
Create a /deploy command that commits, pushes, and opens the live site. Create a /audit command that checks the site for broken links and missing meta tags.
LESSON 06
Hooks & Automation
Settings.json hooks fire at key moments — SessionStart, PreToolUse, PostToolUse. They automate behaviors without manual intervention. The NextElite startup banner is a real example: a hook that runs on every session start to display project status.
EXERCISE
Add a PostToolUse hook that plays a sound when a file is written. Add a SessionStart hook that shows git status on launch.
LESSON 07
MCP Server Integration
MCP (Model Context Protocol) servers extend what Claude Code can do. They add external tools — web search, databases, APIs, file systems — accessible through the same interface. Install servers to give Claude Code new capabilities without custom code.
EXERCISE
Install a web search MCP server and use it to research competitor biohacking sites from within Claude Code.
LESSON 08
GitHub Integration
Claude Code integrates deeply with GitHub. Create PRs with proper descriptions, review code, manage CI/CD — all from the terminal. The gh CLI commands work seamlessly. Automated code review catches issues before they ship.
EXERCISE
Create a feature branch for the NextElite site, make a change, and use Claude Code to create a PR with a proper description.
